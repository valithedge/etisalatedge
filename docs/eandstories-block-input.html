EandStories Block Input Documentation

Block Configuration (dataset attributes)

Attribute Name Purpose Example Value Required
titleType HTML element for title "h2" (default), "h1", "h3" No
articleTitle Title text to display "Latest Stories" No
searchDisabled Hide search form if "true" "true" or "false" No
searchCategoryResultLbl Text for select option "All Categories" No
searchCategoryLbl Text for floating label "Select Category" No

Row/Column Input Structure

Column Index Content Format Required
0 Category HTML text Yes
1 Title HTML text Yes
2 Date HTML text No
3 Description HTML text No
4 Learn More Link or text No
5 Image Picture/Image tag No

Example Input HTML

<div class="eandstories" 
    data-article-title="Latest Stories" 
    data-search-category-lbl="Select Category" 
    data-search-category-result-lbl="All Categories">

    <!-- Row 1 -->
    <div>
        <div>Technology</div>                <!-- col 0: category -->
        <div>New Features Launch</div>       <!-- col 1: title -->
        <div>November 3, 2025</div>         <!-- col 2: date -->
        <div>Exciting new features...</div>  <!-- col 3: description -->
        <div><a href="/news/1">Read More</a></div>  <!-- col 4: learn more -->
        <div>                               <!-- col 5: image -->
            <picture>
                <source srcset="mobile.jpg" media="(max-width: 767px)">
                <source srcset="tablet.jpg" media="(max-width: 991px)">
                <img src="desktop.jpg" alt="Feature Image" class="cover">
            </picture>
        </div>
    </div>

    <!-- Row 2 -->
    <div>
        <div>Business</div>
        <div>Q4 Results</div>
        <div>November 2, 2025</div>
        <div>Financial highlights...</div>
        <div><a href="/news/2">Learn More</a></div>
        <div>
            <img src="business.jpg" alt="Business Chart" class="cover">
        </div>
    </div>
</div>

Important Notes

  • Each row becomes a tile in the output.
  • The Load More button appears automatically when rows.length > 6.
  • All HTML content in columns is preserved (safe HTML allowed).
  • Images support both <picture> and <img> tags.
  • Images are automatically set to lazy-load.
  • Learn More links can be full <a> tags or plain text (converts to link).
  • The category text is used both in the tile header and as a data attribute for filtering.
  • HTML in the category text is stripped when used as a data attribute.
